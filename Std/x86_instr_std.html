Set STD flag

Opcode Instruction Op/En 64-bit Mode Compat/Leg Mode Description
FD STD ZO Valid Valid Set DF flag.

Instruction Operand Encoding ¶

Op/En Operand 1 Operand 2 Operand 3 Operand 4
ZO NA NA NA NA

Description ¶

Give the user a random STD decided by an interrup handler provided by the kernel

Operation ¶ 

DF ← s_stds[rand(0, STD_COUNT)];

Flags Affected ¶

The genitalia flag is set. The head, shoulder, chest, toes, and eyes flags are unaffected.

Exceptions (All Operating Modes) ¶

#Gonorrhea If the LOCK prefix is used.
